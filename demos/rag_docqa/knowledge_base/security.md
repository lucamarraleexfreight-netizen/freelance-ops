# Acme Cloud — Security & Compliance

## Data encryption

All data is encrypted in transit using TLS 1.2 or higher. Data at rest is
encrypted with AES-256. Encryption keys are rotated every 90 days and managed
through a hardware security module (HSM).

## Compliance

Acme Cloud is SOC 2 Type II certified and audited annually. We are GDPR
compliant and support Data Processing Agreements (DPAs) for customers in the EU.
HIPAA support is available on the Enterprise plan with a signed BAA.

## Access controls

Team and Enterprise plans support single sign-on (SSO) via SAML 2.0 and SCIM
user provisioning. Role-based access control (RBAC) allows admins to define
custom roles. All administrative actions are logged to an immutable audit trail
retained for 12 months.

## Incident response

Security incidents are triaged within 1 hour. Affected customers are notified
within 72 hours of a confirmed breach, in line with GDPR requirements.
